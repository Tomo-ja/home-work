How to returns the type of numbers
in a given array with JavaScript

References: Create a function that will receive an array of numbers as an argument and return a new array with distinct elements.


In this article, I'm introducing how to return what kind of numbers appears in a given array, omit the numbers appear after second time.

How the function would work

We will use "forEach" function for a given array as well as "find" function to check whether a number appears in the array more than 2 times.

Quick answer

							
function varietyOfNumber(originalArray) {
	let resultArray = [ ]
	originalArray.forEach( num => {
		if ( !resultArray.find( value => value === num )) {
			resultArray.push( num )
		}
	});
	return resultArray
}

What's happening in the function

  1. Receive a array having some numbers as an argument
  2. Create a empty array that end up returning as a result
  3. Pick up every single number from the given array one by one
    with "forEach" function
  4. Check whether or not the number picked up is in the result array
  5. If NO, add the number into the result array
  6. If YES, do nothing and proceed to next number in given array
  7. Return the result array

Explain "forEach" function

This function is prepared by JavaScript itself, applying for only Array type of object. To use this, you can simply add ".forEach" after an array object. Following "forEach", you set a function with argument inside of "( )", which is what you want to do each element on.

The each number going to be assigned temporary valuable (in the answer function, "num") and be processed by following a function inside of "{}" as syntax of JavaScript ES6.

Explain "find" function

This function is also prepared by JavaScript, applying for only Array type of object. Same as "forEach" function, you can add it after an array. However, what it does is not same as "forEach" whatsoever.

As name of function indicates, it returns whether an element that you set is in an array or not. If there is, then returns that element, by the way it returns first matched element if there are more than 2 elements that is matched. If there is not such an element, then returns undefined, which is considered as false in JavaScript.

Inside of "( )", you set temporary valuable that would be assigned each element in a array you want to search on and the key word that you want to find.In our case, "value" is temporary one, "===" is the comparison operator meaning both of them are completely same, and "num" is the key word that is assigned the each number came from "forEach" function
